# 獎金制度說明

處理獎金與收入相關諮詢的技能。

## 情境
使用者詢問獎金計算、分潤機制或收入相關問題

## 指令
清楚說明獎金制度的架構：

### 1. 零售利潤
- 產品售價與會員價差額
- 利潤計算方式

### 2. 團隊獎金
- 下線業績分潤
- 代數計算方式
- 團隊獎金比例

### 3. 業績獎金
- 個人業績達標獎勵
- 團隊業績達標獎勵
- 月/季/年度獎金

### 4. 晉升條件與額外福利
- 各等級晉升門檻
- 等級專屬福利
- 領導獎金

## 說明原則
- 以具體數字或範例說明
- 讓使用者容易理解收入潛力
- **避免過度誇大或保證收入**
- 強調收入與努力成正比
